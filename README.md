## 通过UICollectionview实现轮播图

### 开发初衷
  - 自己项目里面用到了轮播图，在网上找了一通，发现各个都不是很好用，自己实现了一个，集合了各个家的优点。支持多张，一张。

### 如何使用
- 文件拖拽   
LeCYCircleScrollView.h  
LeCYCircleScrollView.m

- Cocoapods   
  pod 'CircelScrollView', :git => "https://github.com/tiandabao/CircelScrollView.git"
