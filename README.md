## 通过UICollectionview实现轮播图

### 开发初衷
  - 自己项目里面用到了轮播图，在晚上找了一通，发现各个都不是很好用，自己实现了一个，集合了各个家的优点。支持多张，一张。

### 如何使用
- 目前还不支持pods添加，那就拖拽两个文件就行了   
LeCYCircleScrollView.h  
LeCYCircleScrollView.m
